# Directorio de Plugins de FacturaScripts

## Descripción

Este directorio contiene una colección de **módulos/plugins actualizados de FacturaScripts**.

## Propósito

Este repositorio sirve como:
- **Repositorio central** de plugins actualizados y mantenidos para FacturaScripts
- **Referencia** para futuras actualizaciones de módulos
- **Punto de partida** para el desarrollo y mejora de plugins

## Para Claude (Asistente de IA)

Cuando se te solicite ayudar con actualizaciones de plugins de FacturaScripts:

1. **Contexto**: Este directorio contiene versiones actualizadas de plugins para el sistema de facturación FacturaScripts
2. **Estructura**: Cada carpeta generalmente representa un plugin individual con su estructura específica
3. **Objetivo**: Ayudar a mantener, actualizar y mejorar estos plugins según las necesidades del proyecto
4. **Consideraciones**:
   - Revisar la estructura existente antes de hacer cambios
   - Mantener la compatibilidad con FacturaScripts
   - Seguir las convenciones de código del proyecto
   - Documentar los cambios realizados

## Uso

Este directorio está pensado para:
- Desarrollo y actualización de plugins
- Mantenimiento de módulos existentes
- Colaboración en el ecosistema FacturaScripts

## Mantenimiento

Mantén este directorio actualizado con las últimas versiones funcionales de los plugins.
